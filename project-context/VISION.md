# Vision

Agenty makes production agent building feel like starting with a great application framework instead of assembling a one-off demo. Its moat is a learning system: every shipped agent can produce evidence, reusable contracts, evals, and carefully reviewed improvements for the next one.

The product has three reinforcing expressions:

1. **Framework:** open contracts, runtime, UI, starters, adapters, and conformance tooling.
2. **Resource:** precise docs, reference implementations, patterns, comparisons, and release evidence.
3. **Agency:** premium design and implementation powered by the same tested system.

Agenty wins when it shortens time-to-trust, not merely time-to-first-chat. Every agent should be inspectable, interruptible, permission-aware, source-backed, testable, and portable across reasonable provider choices.

It is not a universal autonomous operating system, a private prompt dump, a model gateway reseller, or a replacement for each product's data and authorization layer.
